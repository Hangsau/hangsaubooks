策略查找

依教學問題篩選，備課時快速找到對應策略

提取練習（回想） Ch.1

讓大腦主動召回資訊，比重讀更能固化長期記憶

+

怎麼做

  1. 講課結束後，請學生蓋上筆記，用自己的話寫下剛才學到的重點。
  2. 兩人一組互相解釋，不看筆記，看看彼此能說出多少。
  3. 這就是提取練習——強迫大腦主動搜尋，神經連結就此固化。
間隔重複 Ch.6

把提取練習分散在數天或數月內，比集中複習效果強得多

+

怎麼做

  1. 不要讓學生只在考前一次大複習，而是每隔一段時間就做一點提取練習。
  2. 設計作業時，加入幾題「以前教過的內容」，讓舊知識持續被提取。
  3. 這樣做能讓程序性路徑的神經連結持續強化，知識不容易遺忘。
圖像組織圖 Ch.6

幫學生建立內在基模，比單純寫筆記更能深度理解教材

+

怎麼做

  1. 先教學生表面概念，再介紹圖像組織圖。
  2. 提供有結構的範本（留空格讓學生填入），協助建立內在基模。
  3. 學生有概念後，讓他們自製組織圖，比較和對照不同概念之間的關係。
  4. 完成後將組織圖應用於討論、寫作或深入研究，鞏固學習成果。
填空講義 Ch.2

提供有結構的大綱，降低工作記憶負擔，讓工作記憶較少的學生也能跟上

+

怎麼做

  1. 上課前準備含有空格的大綱或講義，讓學生在聽課時填入關鍵詞。
  2. 控制講課和板書速度，不要讓學生邊抄邊跟不上思路。
  3. 給提示：「我們會看到五個項目，第一項是……」，幫助學生預期結構。
課中暫停 Ch.2

定期暫停讓學生複習筆記，對工作記憶容量少的學生特別關鍵

+

怎麼做

  1. 每講一段新內容後，給學生一兩分鐘整理筆記或向學伴確認理解。
  2. 暫停時提出開放式問題，讓學生兩人一組討論三十秒。
  3. 這讓海馬迴有機會把資訊卸載到新皮質，不讓工作記憶超載。
交替學習 Ch.6

在練習中混合相似但容易搞混的概念，比反覆練習同一種題型效果更好

+

怎麼做

  1. 出作業時，不要只練剛教的內容——混入以前學過、容易混淆的題目。
  2. 向學生解釋「強壯腿」（已知教材）和「虛弱腿」（新教材）的概念，讓他們兩者交替練習。
  3. 這種有益的難度雖然讓學生感覺比較費力，但建立的神經連結更堅固。
思考—配對—分享 Ch.3

讓每位學生都有獨自思考的時間，再透過對話提取和強化記憶

+

怎麼做

  1. 提出問題後，給學生一兩分鐘獨自思考（這個安靜時間讓海馬迴卸載）。
  2. 兩人配對，分享各自的答案。
  3. 全班分享，讓更多不同的理解角度浮現。
一分鐘歸納 Ch.3

請學生用自己的話寫下剛才學的內容，強迫主動提取而非被動接收

+

怎麼做

  1. 講課中途暫停，請學生用一分鐘寫下「剛才學到什麼」。
  2. 或請學生寫下「最困惑的地方」，讓你掌握哪裡需要再說明。
  3. 這個動作讓資訊從工作記憶轉向長期記憶，效果遠比重讀好。
很快輪過一圈 Ch.3

全班依序輪流回答開放式問題，確保每位學生都主動提取，沒有人能躲

+

怎麼做

  1. 提出一個開放式問題（答案多樣，不是對或錯）。
  2. 給學生一分鐘思考或寫下答案，確定每個人都準備好了。
  3. 點名後沿著座位順序輪流，每人說幾個字或一句話，不允許打斷。
  4. 不要即時糾錯——等所有人都說完，再整體回應。
同儕教學 Ch.3

請學生「教」給學伴聽，是最強力的提取練習之一

+

怎麼做

  1. 請一位學生向學伴解釋剛才教的概念，然後互換角色。
  2. 必要時讓學生角色扮演，例如讓學生演出地球繞太陽旋轉。
  3. 確保每位學生都扮演過「教師」的角色，不只是聽別人說。
番茄鐘工作法 Ch.4

25分鐘專注 + 5分鐘放鬆，讓大腦在發散模式中持續處理難題

+

怎麼做

  1. 移除所有干擾（關掉手機通知）。
  2. 計時25分鐘，期間完全專注於一件事。
  3. 放鬆5分鐘：散步、閉眼、喝水——不要滑手機（會蓋掉剛存入的記憶）。
  4. 年紀較小的學生，可調整為「年齡＋1」分鐘，例如9歲→10分鐘。
引起動機的誘因 Ch.10

課程開頭設計讓學生期待意想不到的驚喜，觸發多巴胺，讓教材黏得更牢

+

怎麼做

  1. 用引文、短實驗、投票、角色扮演等方式，讓學生在課程開頭就被吸引。
  2. 確保誘因連結到隨後要教的重要概念，不能只是好玩。
  3. 多樣化：不要每堂課用同一種誘因，意想不到才能持續觸發多巴胺。
  4. 讓每位學生都參與（例如請大家寫下答案），不要只叫少數人回答。
意想不到的獎賞 Ch.7

可預期的獎賞建立動機；意想不到的獎賞激發多巴胺，讓神經連結更容易形成

+

怎麼做

  1. 不要讓獎賞完全可預期——偶爾在意外時機給予讚美或小獎勵。
  2. 注意獎賞的量，太多讚美會讓它失去「意想不到」的效果。
  3. 了解學生追求的獎賞可能是社交認可，不一定是成績——針對這一點設計誘因。
直接教學三步驟 Ch.7

用「看我做→我們一起做→你來做」建立課室程序，直到變成自動習慣

+

怎麼做

  1. 第一步「看我做」：示範你希望學生做的行為或程序，說明清楚。
  2. 第二步「我們一起做」：和學生一起練習，適時給予讚美和修正。
  3. 第三步「你來做」：讓學生獨立練習，直到熟練到不用思考。
  4. 從第一天就開始教，必要時重複強化。
座號排隊收發作業 Ch.7

為日常程序建立程序性記憶，節省教學時間並讓學生有目的地活動

+

怎麼做

  1. 為每位學生分配固定座號（對應成績登記冊順序），請學生記在筆記本封面。
  2. 練習按座號順序排隊收發作業，多練幾次直到三十秒內完成。
  3. 這讓學生離開座位有目的和秩序，同時節省課堂時間，也建立程序性記憶。
合作學習結構 Ch.8

有結構的小組合作讓每位學生負有責任，防止搭便車，同時緩解壓力

+

怎麼做

  1. 制定明確的角色分配（紀錄者、發言者、計時者等）。
  2. 說明什麼是「搭便車」行為，以及你如何發現和處理。
  3. 小組規模：需要規模→較大的組；需要創意→較小的組（2-3人）。
  4. 定期檢視小組合作歷程，不只看成果。
多媒體原則 Ch.9

同時讓學生看到和聽到教材，是線上教學效果最好的方式

+

怎麼做

  1. 製作影片時，確保學生同時能看到畫面和聽到你的聲音解說。
  2. 影片控制在五分鐘左右，其中至少一兩分鐘要看到你本人在說話。
  3. 投資一支好麥克風，注意打光——音質和畫質直接影響學生是否願意看完。
學生製作說明影片 Ch.9

請學生為某個主題製作影片，強迫他們深度理解才能解釋清楚

+

怎麼做

  1. 出「製作說明影片」的作業，讓學生選擇一個概念並解釋給觀眾看。
  2. 使用免費工具（TechSmith Capture、iMovie等），降低技術門檻。
  3. 評分標準要聚焦在概念的正確性，不是影片製作的精緻度。
  4. 把優秀的說明影片存檔，未來可重複使用給新學生看。
