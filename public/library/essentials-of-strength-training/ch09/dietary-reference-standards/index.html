飲食參考標準 Dietary Reference Standards

DRI 定義

Dietary Reference Intakes；美國食品與營養委員會制定；涵蓋巨量＋微量營養素＋電解質＋水；基於慢性病預防，非只是預防缺乏

RDA

Recommended Dietary Allowance：滿足同年齡/性別大多數健康人每日需求的平均攝取量

AI

Adequate Intake：無法建立 RDA 時使用的建議攝取量（例如水分、纖維、鉀）

UL

Tolerable Upper Intake Level：不造成不良影響的最高每日攝取量（含食物＋水＋補充品）；超過 UL 風險增加

EAR

Estimated Average Requirement：滿足同年齡/性別 50% 健康人需求的平均每日攝取量；用於評估群體

AMDR

Acceptable Macronutrient Distribution Range；蛋白質：成人 10–35%；碳水：成人 45–65%；脂肪：成人 20–35%

MyPlate

USDA 飲食指南視覺工具，5 大食物群（蔬菜、水果、穀物、蛋白質、乳製品）。以身體活動量少者為基準，運動員需調整

排除食物群的風險

不喝奶 → 鈣、鉀、維生素 D 不足；不吃動物食品 → B12 不足（需強化食品或補充）；排除整個食物群應諮詢運動營養師
