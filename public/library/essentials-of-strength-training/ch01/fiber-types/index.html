肌纖維類型 Fiber Types

Type I（慢縮）

小運動神經元、低閾值、慢速、抗疲勞高、有氧酶↑、微血管密度↑、紅色

Type IIa（快縮a）

中間型；有氧代謝>IIx；抗疲勞中等；適合混合運動如拳擊/摔角

Type IIx（快縮x）

大運動神經元、高閾值、快速、易疲勞、無氧酶↑、白色；爆發力最強

招募順序

低強度→I型；中強度→IIa；高強度/爆發→IIx（依需求遞增）

運動分布範例

100m衝刺=高II型；馬拉松=高I型；足球/籃球=I+II均高

姿勢肌 vs 動作肌

比目魚肌（姿勢）=高I型；股四頭肌（動力）=I+II混合

分類方法

Myosin ATPase染色（常用）或定量肌球蛋白重鏈（MHC）

提升力量產出

①重負荷訓練優化神經招募 ②增大肌肉截面積 ③多關節爆發性動作
