心血管系統 Cardiovascular
心臟結構
兩個泵：右→肺循環；左→體循環。各有心房（atrium）+心室（ventricle）
心臟瓣膜
AV瓣（三尖/二尖）防止血液從心室逆流回心房；半月瓣（主動脈/肺動脈）防止心室放鬆時逆流
傳導系統
SA node（起搏器）→心房→AV node（延遲）→AV束→束支→Purkinje纖維→心室
心血管系統 Cardiovascular
心率調節
交感→心率↑；副交感→心率↓。正常60-100bpm；<60=心搏徐緩；>100=心搏過速
ECG波形
P波=心房去極化；QRS複合波=心室去極化；T波=心室再極化
血管類型
動脈（離心）→小動脈（調節流量）→微血管（交換）→小靜脈→靜脈（回心）
血液功能
O₂由血紅素運至組織；CO₂帶回肺部；紅血球含碳酸酐酶協助CO₂排除
骨骼肌幫浦
收縮肌肉壓縮靜脈→單向瓣膜推血回心；運動後持續活動避免血液積聚

本體感覺器 Proprioception
本體感覺器
位於關節、肌肉、肌腱，感應壓力與張力，傳遞動覺資訊至CNS
肌梭 Muscle Spindle
平行於梭外纖維；偵測肌肉長度與長度變化速率
牽張反射
肌肉拉長→肌梭激活→脊髓→運動神經元→肌肉收縮（如膝跳反射）
本體感覺器 Proprioception
GTO 位置
位於肌腱近肌腱接合處（myotendinous junction），與梭外纖維串聯
GTO 功能
張力↑→GTO激活→抑制性中間神經元→運動神經元被抑制→肌肉放鬆
Spindle vs GTO
Spindle = 促進收縮；GTO = 抑制收縮（保護機制）
訓練適應
重訓可訓練大腦皮質抑制GTO的能力，為基本神經適應之一
梭內纖維
Intrafusal fibers = 肌梭內改良的肌纖維；梭外纖維 = 一般肌纖維

肌肉結構 Muscle Structure
Epimysium
最外層纖維結締組織，包覆整塊肌肉，與肌腱連續
Perimysium
包覆肌束（fasciculi），每束最多150條肌纖維
Endomysium
包覆單條肌纖維，與肌膜（sarcolemma）連續
肌肉結構 Muscle Structure
肌纖維
直徑50-100μm，多核，橫紋外觀；肌漿（sarcoplasm）含肌原纖維、粒線體
肌節 Sarcomere
最小收縮單位，靜息約2.5μm。Z線為邊界；A帶=myosin；I帶=只有actin；H帶=只有myosin
Myosin（粗絲）
16nm，含球形頭（可形成橫橋）、鉸鏈點、纖維尾
Actin（細絲）
6nm，雙螺旋；含Troponin（結合Ca²⁺）與Tropomyosin（遮蓋結合位點）
T管 & SR
T-tubule傳遞動作電位至深層；SR儲存Ca²⁺，收縮時釋放

肌纖維類型 Fiber Types
Type I（慢縮）
小運動神經元、低閾值、慢速、抗疲勞高、有氧酶↑、微血管密度↑、紅色
Type IIa（快縮a）
中間型；有氧代謝>IIx；抗疲勞中等；適合混合運動如拳擊/摔角
Type IIx（快縮x）
大運動神經元、高閾值、快速、易疲勞、無氧酶↑、白色；爆發力最強
肌纖維類型 Fiber Types
招募順序
低強度→I型；中強度→IIa；高強度/爆發→IIx（依需求遞增）
運動分布範例
100m衝刺=高II型；馬拉松=高I型；足球/籃球=I+II均高
姿勢肌 vs 動作肌
比目魚肌（姿勢）=高I型；股四頭肌（動力）=I+II混合
分類方法
Myosin ATPase染色（常用）或定量肌球蛋白重鏈（MHC）
提升力量產出
①重負荷訓練優化神經招募 ②增大肌肉截面積 ③多關節爆發性動作

呼吸系統 Respiratory
解剖路徑
鼻腔（暖化/濕化/淨化）→氣管→支氣管→細支氣管（23代）→肺泡
吸氣機制
橫膈膜收縮→胸腔負壓→空氣進入；助吸肌：外肋間肌、胸鎖乳突肌、前鋸肌、斜角肌
呼氣機制
安靜時橫膈膜放鬆+肺彈力回縮；用力時腹肌+內肋間肌主動收縮
呼吸系統 Respiratory
氣體交換
擴散原理：O₂從肺泡（高）→微血管（低）；CO₂反向；肺泡O₂分壓高於肺微血管約60mmHg
能量消耗
安靜時呼吸耗能3-5%；劇烈運動可達8-15%（尤其有氣道阻力者）
肺泡壓
吸氣時低於大氣壓→氣流入肺；呼氣時高於大氣壓→氣流出肺
肋膜壓
胸膜與肺膜之間，正常為負壓，協助維持肺擴張
訓練呼吸肌
耐力訓練＋重訓均訓練呼吸肌；一般不需特別訓練，除非術後或長期臥床

神經肌肉系統 Neuromuscular
運動單位
一個運動神經元＋其所支配的所有肌纖維；同一單位內纖維類型相同
神經肌肉接合
乙醯膽鹼（ACh）跨突觸擴散→激發肌膜產生動作電位
All-or-None原則
運動單位要嘛完全收縮，要嘛不收縮；更強的刺激不產生更強的單次收縮
神經肌肉系統 Neuromuscular
Twitch（單收縮）
單一動作電位→短暫力量峰值；Ca²⁺在達峰前即被泵回
Tetanus（強直）
高頻率刺激→twitches疊加→Unfused tetanus→Fused tetanus（最大力量）
力量分級①
頻率調節：提高同一運動單位的激發頻率（小肌肉為主）
力量分級②
招募：啟動更多運動單位（大肌肉為主，如股四頭肌）
精確度 vs 力量
眼肌：1條纖維/神經元（高精確）；股四頭肌：數百條纖維/神經元

骨骼系統 Skeleton
軸心骨骼 Axial
頭顱、脊椎（C1-尾骨）、肋骨、胸骨 ≈ 80塊
附肢骨骼 Appendicular
肩帶（肩胛＋鎖骨）、上肢、骨盆帶、下肢 ≈ 126塊；全身約206塊
纖維關節 Fibrous
幾乎不能動，如顱縫（sutures）；靠纖維結締組織連接
骨骼系統 Skeleton
軟骨關節 Cartilaginous
有限度活動，如椎間盤（intervertebral disks）；靠軟骨連接
滑液關節 Synovial
活動最大；關節面覆透明軟骨（hyaline cartilage）+ 滑液（synovial fluid）；單軸=肘，雙軸=踝/腕，多軸=肩/髖
脊椎區段
頸（Cervical）7 + 胸（Thoracic）12 + 腰（Lumbar）5 + 薦（Sacral）5融合 + 尾（Coccygeal）3-5
骨骼功能
提供槓桿、支撐、保護；肌肉只能拉（pull）不能推（push），透過骨骼槓桿傳遞力量
骨骼適應
重訓/爆發力訓練 → 骨密度↑、骨礦物質↑；適應期比肌肉慢，需變化頻率/強度/類型

滑動絲理論 Sliding Filament
① ATP分解
Myosin ATPase 分解ATP→ADP+Pi，myosin頭「充能」就位
② 橫橋形成
Ca²⁺與Troponin結合→Tropomyosin位移→暴露actin結合位點
③ Power Stroke
釋放Pi→myosin頭彎曲拉動actin向肌節中心→釋放ADP
滑動絲理論 Sliding Filament
④ 解離
新ATP結合myosin頭→與actin解離，頭回復原位
力量決定因素
任一瞬間與actin結合的橫橋數量 = 產生的力量
放鬆條件
運動神經停止 → Ca²⁺泵回SR → Tropomyosin遮蓋結合位點
收縮需要
Ca²⁺（觸發）+ ATP（動力與解離）+ ATPase（催化）缺一不可
形態變化
收縮時：H帶↓、I帶↓、Z線靠近；A帶長度不變
