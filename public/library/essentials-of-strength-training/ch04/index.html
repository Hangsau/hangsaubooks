生長激素 Growth Hormone
22 kDa 單體
191 胺基酸多肽；由前腦垂體 somatotroph 產生；最常被測量的形式（RIA/ELISA）；但非唯一形式
GH 超家族
含多種 splice variants 及聚合體（aggregates）；生物活性 GH 濃度遠高於 22kDa 單體；超家族才是真正作用主體
主要生理功能
↑蛋白合成·↑胺基酸轉運·↑脂肪利用(脂解)·↓葡萄糖利用·↓糖原合成·↑膠原蛋白合成·促軟骨生長
生長激素 Growth Hormone
刺激 GH 分泌的訓練條件
高乳酸 + 低 pH 是主要刺激；10RM（重）+ 短休息（1分鐘）+ 多組數 = 最大 GH 反應；長休息（>3分）GH 反應↓
女性 GH
女性 22kDa GH 靜息濃度高於男性（分泌頻率與振幅更高）；月經週期各期反應不同；生物活性GH隨訓練增加靜息值
分泌模式
脈衝式分泌（pulsatile）；夜間睡眠時 22kDa 最高；運動可增加脈衝振幅與次數；生物活性 GH 無明顯日夜節律
訓練適應
訓練後對同等絕對強度的 22kDa GH 反應↓（效率提升）；生物活性GH靜息值可能↑；長期適應仍在研究中
調節機制
下視丘 GhRH（+）→ 促進 GH；生長激素抑制素 Somatostatin（−）→ 抑制 GH；複雜神經內分泌回饋迴路

皮質醇 Cortisol
來源與分類
腎上腺皮質分泌的糖皮質素（glucocorticoid）；由 ACTH（前腦垂體）刺激分泌；人體主要糖皮質素
分解代謝作用
① 胺基酸→碳水化合物（糖質新生）② ↑蛋白水解酶 ③ ↓蛋白質合成 ④ 抑制糖原合成 ⑤ 抑制免疫細胞功能
日夜節律
早晨濃度最高，全天下降；測量時必須記錄採血時間點；比較研究需控制日夜節律變數
皮質醇 Cortisol
急性 vs 慢性
急性↑：協助去除受損蛋白質，是肌肉重塑過程的一部分；慢性高濃度：負面，導致肌肉萎縮、力量↓
T/C 比值
睾固酮/皮質醇比值：評估合成/分解代謝狀態的指標；皮質醇>800 nmol/L 可能代表過度訓練
升高皮質醇的訓練條件
大訓練量 + 大肌群 + 短休息 → 皮質醇↑；與 GH 升高的訓練條件相似（高代謝壓力）
肌纖維差異
對 Type II 纖維分解代謝效果更強（蛋白質含量更多）；Type I 纖維主要靠減少降解來達到肥大
免疫系統影響
使免疫細胞（B/T cells）失活；運動後免疫抑制的部分原因；影響骨骼肌的恢復與重塑

兒茶酚胺 Catecholamines
來源
腎上腺髓質（medulla）直接由神經刺激分泌；腎上腺素（Epinephrine）為主 + 正腎上腺素 + 多巴胺
生理功能
↑力量輸出（中樞）·↑肌肉收縮速率·↑血壓·↑能量可用性·↑肌肉血流（血管擴張）·促進其他合成荷爾蒙分泌
腎上腺素與乳酸
腎上腺素濃度與乳酸相關；高強度短休息訓練後 NE/E/DA 維持升高 5 分鐘進入恢復期
兒茶酚胺 Catecholamines
力量維持
賽前/訓練前更高兒茶酚胺釋放 → 整個訓練過程中更能維持力量輸出
腎上腺素枯竭
訓練不變化 → 持續應激 → 腎上腺髓質持續參與 → 最終導致腎上腺素能耗竭（adrenergic exhaustion）
Proenkephalins
腎上腺髓質也分泌 Peptide F 等前腦啡肽片段；增強免疫細胞功能；對運動後恢復至關重要
訓練適應
長期重訓 → 最大運動時腎上腺素分泌能力↑；對單次訓練的腎上腺素反應可能↓（效率提升）
胺類特性
由酪氨酸（tyrosine）合成；與胜肽荷爾蒙類似與膜受器結合作用；但不像胜肽荷爾蒙受到直接負回饋調節

受器機制與調節 Receptor Mechanisms
上調 Upregulation
受器數量↑ 或 結合敏感性↑；1~2 次重訓即可增加雄激素受器數量；使現有睾固酮可被更充分利用
下調 Downregulation
過度刺激（如胰島素抵抗）或已達肌肉最大尺寸 → 受器反應性↓；荷爾蒙無法有效與受器結合
雄激素受器
睾固酮的受器；訓練影響最大數量（非結合敏感性）；女性受器上調速度更快（約1小時）
受器機制與調節 Receptor Mechanisms
受器 vs 血中濃度
血中荷爾蒙濃度 ≠ 細胞效果；重要的是受器狀態；低量訓練可能不改變血中T但增加受器數量
神經活化影響
神經活化肌纖維 → 增加肌肉受器結合親和力；訓練水平影響神經-內分泌整合方式
SHBG 作用
性荷爾蒙結合球蛋白可與膜受器結合 → 啟動 cAMP 途徑；結合蛋白本身具有生物活性；不只是運送工具
mTOR 途徑
皮質醇可阻斷 Akt/mTOR 信號通路（mRNA 翻譯起始）；睾固酮反效果；兩者競爭對 DNA 元件的結合
訓練前營養
訓練前攝取蛋白質+碳水化合物可上調骨骼肌雄激素受器含量；進一步強化睾固酮利用效率

訓練處方操控策略 Program Design
最大化睾固酮
① 大肌群動作（深蹲/硬舉/爆發上舉）② 85~95% 1RM ③ 多組數 or 多動作（高容量）④ 短休息 30~60秒
最大化 GH（22kDa）
① 高強度（10RM）+ 三組 ② 短休息（1分鐘）③ 訓練前後補充蛋白質+碳水化合物；關鍵：產生高乳酸+低pH
優化腎上腺反應
大訓練量+大肌群+短休息 → 腎上腺應激；但必須週期化：長休息/低量/完全休息日 → 預防腎上腺枯竭
訓練處方操控策略 Program Design
週期化原則
訓練必須變化（週期化）才能讓腎上腺恢復；避免慢性高皮質醇；防止非功能性超量訓練（overreaching）
急性變項的重要性
強度·組數·動作順序·休息時間·動作選擇 → 決定荷爾蒙反應模式；不同組合產生不同合成/分解代謝環境
募集原則
只有被活化的肌纖維才接受荷爾蒙適應；募集越多運動單位 → 全肌肉重塑範圍越大
過度訓練警示
皮質醇持續>800 nmol/L → 過度訓練警示；T/C比↓；負向蛋白代謝平衡；免疫抑制；肌肉萎縮
營養時機
訓練前後補充蛋白質+碳水化合物可：① 提升 IGF-I ② 上調雄激素受器 ③ 優化合成代謝環境

荷爾蒙基礎概念 Endocrinology
荷爾蒙定義
化學訊號分子；由內分泌腺合成、儲存、分泌入血液；作用於含有特異性受器的目標組織
三大類別
① 類固醇（脂溶性，擴散進細胞）② 胜肽/多肽（水溶性，作用於膜受器）③ 胺類（酪氨酸或色氨酸衍生）
分泌途徑
內分泌（入血）· 旁分泌（鄰近細胞）· 自分泌（作用自身）· 胞內分泌（細胞內）
荷爾蒙基礎概念 Endocrinology
結合蛋白
血液中攜帶荷爾蒙；延長半衰期；防止降解；多數荷爾蒙必須「游離」才有活性（free hormone hypothesis）
鎖鑰理論
受器=鎖；荷爾蒙=鑰匙；每種荷爾蒙對應特異受器；但現知互動比鎖鑰更複雜（交叉反應、異位結合）
類固醇作用機制
擴散穿過細胞膜 → 與細胞質受器結合 → H-RC 移入細胞核 → 與 DNA 結合 → 轉錄 mRNA → 蛋白質合成
胜肽作用機制
與膜受器結合 → 活化二級訊使（cAMP、JAK/STAT 等）→ 訊號傳導至細胞核或代謝途徑
一般適應症候群
Selye 提出；壓力 → 警戒反應（功能下降）→ 抵抗期（適應，功能超越基準）→ 週期化理論基礎

睾固酮 Testosterone
來源與功能
男：睾丸為主；女：卵巢＋腎上腺；促進蛋白質合成·抑制蛋白質分解·合成代謝最重要荷爾蒙
游離睾固酮
僅佔總量 0.5~2%；重阻訓可急性增加游離睾固酮；年齡↑ → 游離睾固酮反應↓；較老年男性生理潛能較小
SHBG
性荷爾蒙結合球蛋白；攜帶睾固酮；可自身與膜受器結合啟動 cAMP 途徑；並非單純載體
睾固酮 Testosterone
男性反應
大肌群運動後血中睾固酮顯著↑；早晨濃度最高；需多組、大重量、足夠肌肉量才能觀察到增加
女性反應
濃度為男性 1/15~1/20；多數研究無顯著急性↑；但雄激素受器上調更快（約1小時）；對少量睾固酮高度敏感
升高睾固酮的訓練條件
① 大肌群動作（深蹲、硬舉、爆發上舉）② 重負荷 85~95% 1RM ③ 多組數/多動作 ④ 中等休息（30~60秒）
神經系統影響
睾固酮與神經元受器結合 → 增加神經傳導物質 → 改變結構蛋白 → 提升神經肌肉力量潛力
訓練適應
長期訓練可小幅增加靜息睾固酮；LH/FSH 同步上升；菁英舉重選手2年訓練後靜息T上升

類胰島素生長因子 IGF
IGF-I 基本
70 胺基酸多肽；主要由肝臟在 GH 刺激後產生；促進蛋白質合成的核心角色；也存在肌肉局部合成
IGF 結合蛋白
至少 6 種（IGFBP-1 ~ IGFBP-6）；IGFBP-1 和 IGFBP-3 研究最多；調節 IGF 的受器可及性；各有獨立生物活性
Mechano Growth Factor
機械刺激（過負荷、伸展）→ 肌肉細胞大量合成 IGF-I；以自分泌方式作用；可能是肌肉 IGF 最重要作用途徑
類胰島素生長因子 IGF
IGF 運動反應
初始濃度低 → 運動後 IGF↑；初始濃度高 → 變化不明顯；需 8~24h 才從肝臟產生；急性增加可能來自細胞破壞
營養的重要性
IGF-I 對急性熱量攝入極敏感；訓練前後補充蛋白質+碳水化合物可顯著提升 IGF-I
局部 vs 循環 IGF
循環中 IGF-I 增加不一定反映肌肉局部 IGF 信號；間質液（ISF）中的 IGF 可能更直接影響肌肉；不同生物隔室
女性 IGF 適應
阻力訓練可增加女性總 IGF-I；降低 IGFBP-1 → 更多 IGF 可與受器結合；顯示運動模式特異性適應
調控因子
GH 為主要刺激；甲狀腺素和睾固酮也參與；營養狀態（正氮平衡、蛋白質攝入）是關鍵調節變項
