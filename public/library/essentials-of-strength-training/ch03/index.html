乳酸與代謝酸中毒
乳酸非疲勞原因
乳酸≠疲勞成因；乳酸脫氫酶反應本身消耗H⁺（鹼化）；實為能量基質
真正酸化來源
ATP水解（ATP→ADP+Pi+H⁺）才是H⁺主要來源；非乳酸堆積；乳酸酸中毒為誤稱
代謝酸中毒效應
抑制糖解酶、干擾興奮-收縮耦合（鈣離子與肌鈣蛋白結合受阻）、抑制橫橋循環
乳酸與代謝酸中毒
乳酸閾值LT
血乳酸開始急劇上升的運動強度；未訓練者50-60% VO₂max；訓練者70-80%
OBLA
血乳酸達4 mmol/L的第二個轉折點；對應大運動單位（Type II）大量徵召
乳酸清除
停止運動後約5分鐘達峰值；1小時內回基線；主動恢復優於被動；有訓練者清除更快
Cori Cycle
乳酸→血液→肝臟→糖質新生（glucose）；長時間運動與恢復期的重要機制
訓練對LT影響
LT/OBLA右移（更高強度才開始堆積）；粒線體增加、兒茶酚胺分泌減少

氧化系統
Krebs Cycle
丙酮酸→乙醯CoA（pyruvate dehydrogenase）→進入TCA；每葡萄糖產2 ATP（GTP）
電子傳遞鏈ETC
NADH→3 ATP；FADH₂→2 ATP；O₂為最終電子受體→形成H₂O；氧化磷酸化
葡萄糖總ATP產量
血糖：38 ATP；肌糖原：39 ATP；氧化磷酸化佔>90%的ATP合成
氧化系統
脂肪氧化
三酸甘油酯→游離脂肪酸→β氧化→乙醯CoA→Krebs；1分子棕櫚酸(16C)可產>100 ATP
蛋白質氧化
胺基酸→葡萄糖/丙酮酸/Krebs中間物；貢獻3-18%（長時間運動）；BCAA為主要氧化胺基酸
基質偏好
靜息：70%脂肪+30%碳水；高強度：近100%碳水；長時間次極量：逐漸轉向脂肪
Krebs限速步驟
isocitrate→α-ketoglutarate（isocitrate dehydrogenase）；ADP活化、ATP抑制
容量最大
ATP生成速率最慢（#5）但總容量最大（#1）；適合低強度長時間運動（>3分鐘）

能量產生與容量
速率vs容量反比
能量系統速率越高，總容量越低；磷酸原最快但量最少；脂肪氧化最慢但量最多
0-6秒活動
極高強度；主要系統：磷酸原；例：鉛球、舉重抓舉
6-30秒活動
非常高強度；磷酸原＋快速糖解；例：100m衝刺
能量產生與容量
30秒-2分鐘
高強度；快速糖解為主；例：400m跑、200m游泳
2-3分鐘活動
中強度；快速糖解＋氧化系統；例：1500m跑
>3分鐘活動
低強度；以氧化系統為主；例：馬拉松、長跑
速率排名
1磷酸原 2快速糖解 3慢速糖解 4碳水氧化 5脂肪/蛋白氧化
三系統同時作用
任何時刻三系統皆同時活化；由強度（主）和持續時間（次）決定各系統貢獻比例

訓練代謝專一性
代謝專一性原則
運動強度與休息間隔可「選擇」特定能量系統；訓練設計需符合運動代謝需求
間歇訓練原理
預設功休比允許更高強度完成更多訓練量；同強度下比連續訓練疲勞更少
功休比指引
90-100%max：1:12-1:20（磷酸原）；75-90%：1:3-1:5（快速糖解）；>3min：1:1-1:3（氧化）
訓練代謝專一性
HIIT定義與效益
短暫高強度+間歇恢復；改善VO₂max、緩衝能力、糖原含量、無氧閾值；時間效率高
HIIT 9大變數
工作強度/持續時間、恢復強度/持續時間、每組重複次數、組數、組間休息、恢復強度、訓練模式
HIIT最佳化
累積時間於>90% VO₂max；同時徵召大運動單位＋近最大心輸出量→氧化纖維+心肌適應
組合訓練風險
有氧訓練可能降低無氧能力；同時訓練可能抑制最大肌力增益（第7週後停滯甚至下降）
組合訓練注意
無氧訓練本身可提升有氧功率與恢復；大量有氧耐力訓練對無氧運動員可能適得其反

基質耗竭與補充
CP耗竭速率
高強度運動5-30秒可下降50-70%；極度衰竭可近乎耗盡；有氧運動耗竭較慢
磷酸原回補時間
ATP完全回補：3-5分鐘；CP完全回補：約8分鐘；主要靠有氧代謝完成
肌糖原儲量
全身肌肉300-400g；肝臟70-100g；可受訓練與飲食影響
基質耗竭與補充
糖原耗竭與強度
>60% VO₂max肌糖原為關鍵能源；阻力訓練可使肌糖原下降20-60%（少數組數即可）
肌糖原回補策略
運動後每2小時攝取0.7-3.0 g/kg碳水；前4-6小時最快回補；24小時可完全回補
血糖維持
<50% VO₂max血糖相對穩定；>90分鐘後可能下降；<2.5 mmol/L可能低血糖反應
訓練對磷酸原影響
有氧訓練可提升靜息磷酸原濃度；阻力訓練5週後triceps brachii磷酸原濃度增加
兒童vs成人CP
兒童氧化能力更強→CP耗竭較少、回補較快、運動後pH較高；更依賴有氧代謝

糖解作用
定義
碳水化合物（肌糖原或血糖）分解再合成ATP；發生在肌漿（sarcoplasm）
快速糖解
丙酮酸→乳酸；快速再生NAD⁺；H⁺增加→pH下降；無氧條件；短時間
慢速糖解
丙酮酸→粒線體→Krebs cycle；有氧條件；速率慢但持續時間長
糖解作用
淨ATP產量
血糖起始：淨2 ATP；肌糖原起始：淨3 ATP（不需hexokinase消耗ATP）
關鍵酶調控
PFK（限速步驟）：ATP抑制、AMP活化；Hexokinase：G-6-P抑制；Pyruvate kinase：ATP抑制
磷酸化方式
基質磷酸化（substrate-level）：4 ATP gross；氧化磷酸化（oxidative）：via ETC
糖解促進因子
ADP↑、Pi↑、AMP↑、NH₃↑、pH輕微下降→糖解加速；pH顯著下降→抑制糖解
糖解終產物
→乳酸（無氧/能量需求高）or 丙酮酸進粒線體（有氧/能量需求低）；由細胞能量狀態決定

磷酸原系統
ATP水解
ATP+H₂O → ADP+Pi+H⁺+能量；由ATPase催化；肌肉儲量僅80-100g
肌酸激酶反應
ADP+CP → ATP+肌酸；快速提供能量，但CP有限，不能長時間供能
腺苷酸激酶反應
2ADP → ATP+AMP；AMP為強力糖解促進劑，啟動糖解加速能量產生
磷酸原系統
CP儲量
靜息時CP為ATP的4-6倍；Type II纖維CP濃度 > Type I；爆發性運動更有效
質量作用定律
反應物/產物濃度驅動反應方向；ADP↑ → 肌酸激酶加速；ATP↑ → 反應減緩
適用活動
0-10秒極高強度；阻力訓練、衝刺；所有運動初始皆高度活化
ATP耗竭上限
肌肉ATP最多下降50-60%（維持基本細胞功能需求）；CP可近乎完全耗竭
能量速率
三系統中ATP生成速率最快（速率排名#1）；但總容量最小（容量排名#5）

攝氧量與EPOC
氧虧 Oxygen Deficit
運動初期有氧系統反應慢→需無氧補充的能量量；強度越高氧虧越大
穩態 Steady State
低強度持續運動數分鐘後，攝氧量等於需氧量；高強度運動無法達到穩態
EPOC定義
運動後超過靜息水準的額外攝氧量；用於使身體恢復至運動前狀態
攝氧量與EPOC
EPOC影響因素
強度>持續時間；>50-60% VO₂max且>40分鐘時EPOC最大；超極量間歇可最大化EPOC
EPOC生理機制
血液/肌肉O₂補充、ATP/CP再合成、體溫↑、三酸甘油酯-脂肪酸循環、蛋白質更新
無氧vs有氧貢獻
0-5秒：96%無氧；30秒：75%無氧；60秒：50/50；90秒：35%無氧；逐漸轉向有氧
阻力訓練EPOC
大重量（80-90%1RM，3組）EPOC > 循環訓練（50%1RM）；阻力訓練也具強度依賴性
高強度EPOC特性
超過VO₂max的高強度運動：整個運動期間持續存在氧虧；無法達穩態
