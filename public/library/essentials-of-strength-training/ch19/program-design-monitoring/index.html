計畫設計與監控 週期化 / COD Deficit

週期化變數

訓練間距、順序、頻率、強度、休息間隔、組數、總量、功重比；與重訓週期化概念相同

速度監控指標

接地時間、步長、步幅、騰空時間、步幅角度、速度、加速度；搭配高速攝影機

COD Deficit

= 505 測驗時間 - 10m 直線時間；排除直線速度影響，反映純 COD 能力

出速與入速監控

Entry velocity（植地前）＋ Exit velocity（離地後第一步）；搭配接地時間判斷 RFD

決策時間

刺激出現到第一確定踩地間距；負值＝預測到刺激前已動（只在真人刺激中出現）

敏捷訓練進程

COD 閉式技能（前進煞車→側向煞車→再加速）→ 加入一般刺激 → 加入運動特異刺激

需求分析步驟

①運動需求分析 ②比較標準分→找強弱 ③確定主次訓練目標 ④分配各 block 時間比例

考試關鍵

★ Illinois test 改善可能只反映代謝能力提升，非 COD 改善（測驗時間過長）
