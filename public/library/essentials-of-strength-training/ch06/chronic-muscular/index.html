慢性肌肉適應 Chronic Muscular Adaptations

粒線體↑

粒線體數量與體積均增加（增加有氧 ATP 生產能力）；配合肌紅素↑更有效利用 O2

肌紅素 (myoglobin)↑

細胞內 O2 運輸蛋白；量增加 → 細胞攝取並利用 O2 能力↑

Type I 纖維肥大

選擇性肥大但截面積不如 Type II 明顯；氧化能力提升（粒線體密度↑）

Type IIx→IIa 轉變

Type II 纖維之間可轉換；IIa 氧化能力優於 IIx，功能特性更接近 Type I

OBLA 上升

訓練後 OBLA 可達 VO2max 的 80–90%；因脂肪利用↑、乳酸生成↓、移除↑、糖原節省

酶活性↑

檸檬酸合酶、琥珀酸脫氫酶等有氧酶↑；糖解酶（如 LDH）可能不變或↓

能源儲存↑

肌肉 glycogen↑、三酸甘油酯↑；ATP 及磷酸肌酸儲量略增

Type I vs II 比較

有氧訓練對 Type I 和 Type II 的有氧潛力相對增加幅度相似；但 Type I 起點更高
