⚡ 神經適應 Neural Adaptations
中樞適應
大腦運動皮質活性↑；沿皮質脊髓束傳遞；可徵召更多快縮運動單位
Size Principle
小→大依閾值順序徵召；快縮纖維需高閾值；爆發性動作可例外選擇性徵召
選擇性徵召
爆發/彈道式動作可跳過慢縮直接徵召快縮（奧舉、增強式、衝刺）
發火率與同步化
↑firing rate → 收縮疊加 → 更大力；多運動單位同步化 → 力量峰值更高
神經
適應
NMJ 適應
突觸面積↑；神經末梢分支↑；乙醯膽鹼受體分散範圍↑ → 傳導效率↑
肌梭反射強化
阻力訓練可提升反射強化 19–55%；訓練者反射電位高於未訓練者
Cross-Education
單側訓練使對側肌力↑最多 22%（平均 8%）；源自中樞神經適應
拮抗肌協同收縮↓
訓練後拮抗肌過多協同收縮↓ → 淨力輸出↑；保護機制與速度需求需平衡

💪 肌肉生長 Muscular Hypertrophy
肥大定義
肌纖維橫截面積（CSA）↑；actin/myosin 蛋白合成↑；myofibril 數量↑
mTOR 路徑
機械負荷激活 Akt/mTOR → 肌蛋白合成↑；訓練後合成率↑維持最多 48 小時
EIMD 與修復
運動誘發肌肉損傷 → 基因表達改變 → 發炎修復 → 蛋白轉換↑ → 長期肥大
機械 vs 代謝刺激
機械：重負荷 + 離心 + 低中量；代謝：高量 + 短休息（健美式）→ 最強合成激素反應
肌肉
生長
Hyperplasia
肌纖維數量↑（縱裂）；動物模型確認；人體中具爭議，若存在佔比 <10%
Type II 肥大更顯著
快縮纖維 CSA 增幅 > 慢縮；快縮比例越高，肥大潛力越大（遺傳影響）
粒線體密度↓
肌肉肥大使體積↑不成比例 → 粒線體密度（per unit volume）相對下降
ATP/CP/肝糖↑
5 個月阻力訓練後：CP ↑28%、ATP ↑18%、肝糖 ↑112%（超補償效應）

🔄 肌纖維轉型 Fiber Type Transitions
轉型方向
無氧訓練：IIx → IIax → IIa（更有氧化能力）；去訓練反向：IIa → IIx
轉型速度
女性 2 週（4 次訓練）、男性 4 週即見顯著 IIx 減少；8 週後 IIx 從 18% → 7%
MHC 改變
IIx MHC → IIa MHC；與睾酮/皮質醇交互作用相關；去訓練可能造成 IIx overshoot
Type I ↔ II
Type I 與 II 之間的轉換幾乎不可能；MHC isoform 差異太大；為遺傳決定
纖維
轉型
有氧+阻力並用
同時進行高強度阻力 + 有氧訓練：IIx → IIa 幾近完全轉換（最大效果）
Pennation Angle
阻力訓練↑羽狀角 → 允許更多蛋白質沉積 → CSA↑；肌束長度↑（sprinter 尤明顯）
衝刺與跳躍訓練
衝刺訓練↑鈣離子釋放 → actin/myosin 橋接↑ → 速度與爆發力↑
緩衝能力↑
高強度間歇訓練可提升緩衝容量 16–38% → 延緩疲勞、提升肌耐力

🦴 結締組織適應 Bone & Connective Tissue
骨骼建模
機械負荷 → osteoblast 遷移 → 分泌膠原蛋白 → 礦化（hydroxyapatite）→ 骨徑↑
MES 閾值
最小必要應變（MES）≈ 骨折力的 1/10；超過閾值才刺激新骨生成；進階超負荷原則
BMD 與肌力相關
阻力訓練者 BMD 高於同齡對照；肌力 / 肌肉量增加伴隨 BMD↑；適應期約 6 個月
骨骼訓練原則
多關節結構性動作（深蹲/爆發上搏）優先；軸向力向量穿越脊椎及髖；1–10RM 負荷
結締
組織
肌腱適應
膠原纖維直徑↑、fibril 數↑、packing density↑；阿基里斯腱硬度 8 週↑15–19%
韌帶與筋膜
高強度負荷刺激膠原合成；肌腱血流差、癒合慢；全ROM動作有助力量傳遞
軟骨
依靠滑液擴散獲取營養；固定化→軟骨萎縮；中等強度運動維持軟骨厚度
膠原蛋白結構
procollagen → 三螺旋 → 微纖維 → 纖維束；cross-linking 提供真正強度

🧪 內分泌反應 Endocrine Adaptations
急性合成激素↑
運動後 testosterone、GH、cortisol 升高可達 30 分鐘；大肌群動作 + 中高量 + 短休息效果最大
GH 與乳酸
血乳酸 / H⁺ 累積是 GH 和 cortisol 分泌的主要刺激因子
IGF-I
GH 的下游媒介；刺激幾乎所有細胞生長（骨骼肌、軟骨、骨）；運動後延遲上升
慢性急性反應改變
長期訓練 → 承受更大負荷 → 急性內分泌反應程度更大（以 GH 最明顯）
內分泌
反應
靜息濃度不變
長期訓練後靜息激素濃度幾乎不變；訓練中短暫↑已足夠刺激組織重塑
AR 受體上調
阻力訓練後 48–72 小時內 AR ↑；高量訓練（6 組）先下調再上調；補充蛋白+碳水可減緩下調
兒茶酚胺
腎上腺素 / 去甲腎上腺素反映無氧訓練急性需求；調節力量輸出與其他激素
受體下調風險
慢性高激素暴露 → 受體下調（如 type 2 diabetes 與胰島素）；類固醇使用者需周期循環

❤️ 心血管反應 Cardiovascular Adaptations
急性反應
HR、SV、CO、BP 全部急性↑；高強度腿推 峰值 BP 達 320/250 mmHg；HR 170 bpm
血壓相位差
向心期 BP 高於離心期（尤其在 sticking point）；離心期 SV/CO 反而較大
血流阻斷
>20% MVC → 局部毛細血管壓閉 → 代謝物累積（H⁺/乳酸）→ 休息期反應性充血
慢性靜息適應
靜息 HR ↓5–12%（短期）；systolic/diastolic BP ↓2–4%；SV 絕對值↑
心血管
反應
左心室壁厚↑
慢性阻力訓練→LV壁厚↑；校正體表面積後差異縮小；與有氧訓練（腔室↑）不同
運動心血管鈍化
訓練後同等絕對強度下 HR/BP/double product 急性反應↓；afterload↓ → CO↑
通氣反應
組間通氣最高（>60 L/min）；短休息（30s）通氣更高；訓練後每分通氣當量↓
有氧阻力並用
大量有氧（頻率/強度過高）可能抑制力量/爆發力增益；合理頻次安排可降低干擾

🔥 過度訓練 Overtraining
連續體概念
Training overload → Acute fatigue → FOR（天~週）→ NFOR（週~月）→ OTS（月~年）
FOR（功能性過度訓練）
計畫性短期表現下降；數天~週內恢復；tapering 後可超補償（計畫性策略）
NFOR（非功能性）
表現停滯/下降持續數週~月；荷爾蒙、心血管、免疫異常；難與OTS區分
OTS 兩型
交感型：靜息交感活性↑（年輕速度/力量型）；副交感型：最終所有狀態都走向此
過度
訓練
荷爾蒙標記
T/C ratio↓；量型OTS：cortisol↑、LH↓、T↓；強度型OTS：靜息激素多不變
心理標記
POMS 惡化：疲憊↑、活力↓、憂鬱/憤怒/焦慮↑；心理變化常先於表現下降出現
OTS 識別清單
不明原因表現↓、持續疲勞、睡眠/食慾紊亂、最大測試退步、訓練量/強度急升
預防策略
漸進超負荷 + 足夠恢復期；監控組合指標（表現+生化+心理）；無單一黃金標記

📉 去訓練 Detraining
肌力維持期
停訓後肌力普遍可維持 4 週；高訓練者離心力與爆發力可能提早下滑
短期影響（≤2週）
14 天停訓：1RM bench –1.7%、squat –0.9%；等速/等長力幾乎不變；跳躍維持
中期影響（8–12週）
肌力僅損失 7–12%；但 EMG↓（神經機制先退）；快縮纖維 CSA 先萎縮
快縮 vs 慢縮萎縮
快縮纖維 CSA 優先萎縮（14天即 –6.4%）；慢縮纖維初期不受影響
去
訓練
長期影響
7 個月：所有纖維均萎縮 ~37%；13.5 月後健美選手無脂肪體重 –9.3%
IIx overshoot
去訓練後 IIa → IIx；可能超過訓練前 IIx 比例（反向過衝現象）
肌肉記憶
恢復訓練時肌力重獲速度快；停訓後肌力幾乎不低於訓練前水準（residual effect）
可逆性原則
訓練適應為暫時性；頻率/量/強度任一嚴重減少皆可引發去訓練效應
